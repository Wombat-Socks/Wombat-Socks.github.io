# Wombat Socks

The landing page for **Wombat Socks**, a tool that turns recipe links and ingredient lists into a populated
[Queen Victoria Market](https://qvm.com.au/shop/) shopping cart.

---

### Run

```bash 

cd wombat-socks && npm install && npm start          # dev at localhost:3000
npm run build:pages                                  # build into ../docs for Pages
```

---

### Deploy

Commit the updated `docs/` folder and push to deploy.
